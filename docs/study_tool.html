Project Overview

This pygame project implements a dynamic square simulation with 50 animated entities that exhibit predator-prey behavior. Smaller squares flee from larger ones, while larger squares chase smaller ones—all within a time-based physics system.

Core Concepts Covered

• Object-Oriented Design: Square class with state management and behavior methods

• Game Loop Architecture: Event handling, physics updates, rendering pipeline

• Time-Based Physics: Delta-time integration for frame-rate-independent movement

• Vector Math: Trigonometric calculations for direction and distance

• Behavioral AI: Flee and chase logic based on size and proximity

• Entity Lifecycle: Lifespan-based respawning and state recycling

• Type Safety: Comprehensive type hints throughout the codebase
